# Images Directory

This directory contains:
- **profile-photo.jpg** - Your professional profile photo (replace with your image)
- **certificates/** - Subdirectory for your achievement certificates

## How to add your image:
1. Add your profile photo to this folder and name it `profile-photo.jpg`
2. The portfolio will automatically display it in the hero section
3. Ensure the image is in JPG, PNG, or WebP format
4. Recommended size: 400x400px or larger for best quality

## How to add certificates:
1. Create a `certificates` subdirectory if needed
2. Add your certificate images (PNG, JPG, or PDF)
3. Update the HTML to reference certificate images in the achievements section
