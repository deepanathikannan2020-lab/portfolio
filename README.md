# Professional Portfolio Website - Deepan Athi K

## 🌟 Overview

A premium, modern, and professional personal portfolio website designed with a futuristic UI/UX, smooth animations, and responsive layout. This portfolio is crafted to impress recruiters, highlight projects, and showcase technical skills.

## ✨ Features

### Design & UI/UX
- **Glassmorphism Design**: Modern glass-effect cards with blur backdrop
- **Futuristic Color Scheme**: Neon blue (#00d4ff), purple (#a855f7), and pink (#ff006e) gradients
- **Smooth Animations**: Fade-in, slide-in, and floating animations throughout
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop devices
- **Interactive Elements**: Hover effects, glow effects, and smooth transitions
- **Dark Theme**: Professional dark background with light text for better readability

### Interactive Features
- 📸 **Profile Photo Upload**: Upload and display your professional photo
- 📄 **Resume Upload & Download**: Upload PDF resume and download functionality
- ✉️ **Contact Form**: Send messages directly from the portfolio
- 🎯 **Scroll Progress Bar**: Visual indication of page scroll progress
- 🔄 **Typing Animation**: Dynamic text cycling in hero section
- 🌐 **Social Links**: GitHub, LinkedIn, and Email integration
- 📊 **Animated Statistics**: Counter animation for achievements

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend server required for basic functionality

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/deepanathikannan2020-lab/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```

## 🎨 Customization

### Update Personal Information

1. **Name & Title** - Edit `index.html` in the hero section
2. **About Section** - Modify the about content and interests
3. **Education Details** - Update education information
4. **Skills** - Add/remove skills and modify proficiency percentages
5. **Projects** - Update project details and links
6. **Contact Info** - Edit email, phone, and social links

### Color Scheme
Modify CSS variables in `css/styles.css`:
```css
:root {
    --primary-color: #00d4ff;      /* Neon Blue */
    --secondary-color: #a855f7;    /* Purple */
    --tertiary-color: #ff006e;     /* Pink */
    --dark-bg: #0a0e27;            /* Dark Background */
    --text-light: #e0e0e0;         /* Light Text */
}
```

### Social Links
Update in `index.html`:
```html
<a href="https://github.com/yourusername" target="_blank">GitHub</a>
<a href="https://linkedin.com/in/yourprofile" target="_blank">LinkedIn</a>
<a href="mailto:your.email@example.com">Email</a>
```

## 🎛 Features Guide

### Photo Upload
1. Click the camera icon in the hero section
2. Select your profile photo
3. Photo will be displayed in the placeholder

### Resume Upload
1. Click "Upload Resume" button
2. Select your PDF file
3. Resume will be displayed in preview
4. Click "Download Resume" to download

### Contact Form
1. Fill in your name, email, and message
2. Click "Send Message"
3. Your default email client will open with pre-filled information

## 🚀 Deployment

### Deploy on GitHub Pages

1. Push to GitHub repository
2. Go to Settings > Pages
3. Select main branch as source
4. Your portfolio will be live at `username.github.io/portfolio`

### Deploy on Vercel

1. Connect your GitHub repository
2. Vercel will auto-deploy on each push
3. Get a custom domain URL

### Deploy on Netlify

1. Connect GitHub repository
2. Set build settings
